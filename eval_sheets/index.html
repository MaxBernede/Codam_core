Contributing

Contributions to this repository are welcome. If you wish to add an evaluation form for a project or improve an existing one, please follow these steps:

  1. Fork the repository.
  2. Clone your fork to your local machine.
  3. Make your corrections or additions.
  4. Commit the changes with a descriptive message.
  5. Push the changes to your fork.
  6. Submit a pull request.

Using the Forms

Students can use these forms to evaluate the performance of each of their projects. They offer a standardized method for assessing projects and form the basis for constructive feedback. To use the forms, simply complete the sections for comments and grading.

Language

You can change the language of the corrections directly in your browser.

Feedback

If you have any feedback or suggestions for improvement, please open an issueor submit a pull request. I'd love to hear from you!
